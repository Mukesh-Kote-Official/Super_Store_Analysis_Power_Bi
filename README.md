# Superstore Sales Dashboard – README

## Project Overview
This Power BI dashboard provides an interactive analysis of Superstore sales data. The report helps users monitor key business metrics, identify top-performing categories, analyze regional sales performance, and track sales trends over time.

## Dashboard Features

### KPI Cards
The dashboard includes the following key performance indicators:

• Total Sales
• Total Profit
• Total Quantity Sold
• Total Discount

### Visualizations

#### 1. Sales by Category (Pie Chart)
Displays the contribution of each product category to total sales.

#### 2. Sales by Region & Sub-Category (Bar Chart)
Shows sales performance across different regions and product sub-categories.

#### 3. Sales Trend Over Time (Line Chart)
Tracks sales performance based on order dates to identify trends and seasonality.

### Interactive Filters (Slicers)

Users can filter the entire dashboard using:

• Region
• Category

## Data Source

Dataset: Superstore Dataset

The dataset contains information such as:

• Order Date
• Region
• Category
• Sub Category
• Sales
• Profit
• Quantity
• Discount

## Business Insights

• Monitor overall business performance.
• Identify high-performing product categories.
• Compare regional sales performance.
• Track sales trends over time.
• Analyze the impact of discounts on sales and profit.

## Tools Used

• Power BI Desktop
• Data Modeling
• DAX Measures
• Interactive Visualizations
• Slicers and Filters

## Dashboard Objective

The objective of this dashboard is to provide a centralized view of sales performance, helping management make data-driven decisions and improve business growth.
